# Contexto del Proyecto: Linkin Park @ Auditorio Miguel Rios - Hoja de Ruta

## Proposito
Este repositorio contiene el dossier operativo digital del show de Linkin Park en el Auditorio Miguel Rios.
Su funcion es consolidar informacion tecnica, logistica y de coordinacion en una web estatica publica por GitHub Pages.

## Arquitectura
- Frontend estatico con HTML + CSS + JavaScript (sin frameworks).
- Navegacion por pestanas en una sola pagina (SPA ligera).
- Contenido operativo cargado desde archivos Markdown por seccion.
- Script de sincronizacion local para transformar Markdown a HTML basico y validar estructura.

## Archivos clave
- /index.html: interfaz principal de consulta.
- /context.md: este documento de referencia.
- /sync_dossier.js: sincronizador local de contenidos Markdown.
- /Sincronizar_Dossier.bat: lanzador rapido en Windows.

## Convencion principal
Este dossier se trata como proyecto independiente. No reutiliza branding ni narrativa de otros recintos o artistas.
